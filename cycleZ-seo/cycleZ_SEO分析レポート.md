## 6. アクションプラン

### Phase 1: 緊急対応（今週中）

#### 1-1. meta descriptionの設定 【最優先】

**推奨文（120文字以内）**
```
岡山駅徒歩5分のロードバイク専門店cycleZ。初心者・女性向けの無料講習会を毎月開催。カウンセリング接客でお客様に合った一台をご提案。GIOS・FELT取扱い。☎086-252-7744
```

**設定方法（WordPressの場合）**
1. 管理画面 → 外観 → テーマエディター
2. header.php または SEOプラグインで設定
3. All in One SEO または Yoast SEO プラグイン推奨

#### 1-2. OG画像の変更 【優先度:高】

**現状**: noimg.png（デフォルト画像）
**推奨**: 店舗外観または女性サイクリストの写真（1200x630px）

SNSでシェアされた際の印象が大幅に改善

#### 1-3. OG descriptionの設定

meta descriptionと同じ内容でOK

**目標**: 月2〜4記事

| 女性がロードバイクを始めるときの完全ガイド | ロードバイク 女性 始め方 | 高 |

| 通勤にロードバイクをおすすめする5つの理由 | ロードバイク 通勤 | 中 |
| 初心者講習会に参加したお客様の声 | - | 中 |

#### 2-2. お客様の声ページの作成 【行動経済学効果:大】

**構成案**
- 初心者から始めた方のインタビュー（3〜5名）
- 女性ライダーの体験談（2〜3名）
- 通勤で使っている方の声（2〜3名）

**行動経済学的効果**
- 「社会的証明」：他の人も選んでいる安心感
- 「類似性」：自分と同じ立場の人が成功している
- 「損失回避」：この店を選ばないと損をするかも

#### 2-3. イベントレポートの更新

2023年で止まっているレポートを最新化
→ サイトが「生きている」印象を与える
